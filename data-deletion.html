Удаление данных и отзыв доступа приложения

Это приложение используется владельцем аккаунта Threads (@svetlanafedorova_psy) для публикации собственного контента через официальный Threads API. Если вы хотите отозвать доступ приложения к аккаунту Threads или запросить удаление связанных с ним данных — воспользуйтесь одним из способов ниже.

Способ 1 — отозвать доступ через настройки Threads

  1. Откройте приложение Threads
  2. Настройки → Аккаунт → Разрешения приложений (Apps and Websites)
  3. Найдите приложение «[ВСТАВЬ НАЗВАНИЕ ПРИЛОЖЕНИЯ]»
  4. Нажмите «Удалить» / «Отозвать доступ»

После этого приложение теряет доступ к аккаунту, токен становится недействительным.

Способ 2 — через настройки Facebook

  1. Facebook → Настройки и конфиденциальность → Настройки
  2. Раздел «Приложения и сайты» (Apps and Websites)
  3. Найдите приложение «[ВСТАВЬ НАЗВАНИЕ ПРИЛОЖЕНИЯ]»
  4. Нажмите «Удалить»

Полное удаление хранимых данных

Отзыв доступа автоматически делает токен недействительным, но если вы хотите, чтобы связанные с приложением данные (тексты, метрики, история правок) были полностью удалены с сервера — напишите на [ВСТАВЬ EMAIL] с темой «Удаление данных Threads», данные удаляются вручную в течение 5 рабочих дней, о завершении сообщим ответным письмом.
